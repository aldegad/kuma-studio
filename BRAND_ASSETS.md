# Brand Assets and Visual Identity

This document covers non-code assets in this repository that are part of Kuma
Studio's visual identity.

## Scope

Unless a file states otherwise, the Apache License, Version 2.0 applies to
source code and code-focused documentation in this repository. It does not
apply to the project's brand assets.

Brand assets include, without limitation:

- character artwork, illustrations, portraits, sprites, and generated
  character images
- logos, icons, badges, and mascot imagery
- product and character identity artwork
- visual branding used to identify the original project

Current examples in this repository include files under:

- `packages/studio-web/public/characters/`

## Character identity

The visual identity of the project's characters, including their names,
presentation, and associated artwork, is part of the project's brand identity.
Copying the code does not grant permission to present those characters as your
own original brand or mascot.

## Separate permissions

If a particular asset is released under a different license or with separate
written permission, that separate grant controls that asset.
