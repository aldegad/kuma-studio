# Code of Conduct

## Our Pledge

We want Kuma Picker to be a friendly place for contributors, users, and curious readers.

Please be respectful, direct, and generous with context. Critique code and ideas, not people.

## Expected behavior

- assume good intent, but be honest
- explain reproduction steps and tradeoffs clearly
- prefer precise feedback over dismissive feedback
- keep performance and benchmark claims factual
- help newer contributors understand the repo instead of gatekeeping it

## Unacceptable behavior

- harassment, intimidation, or personal attacks
- bad-faith benchmark spin or deliberately misleading claims
- publishing private security details before a fix is available
- dismissing questions because they seem basic

## Reporting

If someone violates this code of conduct, email `aldegad@gmail.com`.

We will review reports in good faith and aim for a proportionate response.
